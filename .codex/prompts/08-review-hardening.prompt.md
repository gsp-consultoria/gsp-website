# Prompt 08 — Code review e hardening

Faça um code review completo da landing page.

## Avaliar

- Arquitetura de pastas.
- Responsabilidade dos componentes.
- Acessibilidade.
- Responsividade.
- SEO.
- Performance.
- Qualidade da copy.
- Tipagem TypeScript.
- Testes.
- CI.

## Saída esperada

1. Lista de problemas encontrados.
2. Correções aplicadas.
3. Itens que exigem decisão do responsável.
4. Comandos executados.

## Critérios de aceite

- Nenhum problema crítico pendente.
- `npm run validate` passa.
- Documentação permanece atualizada.
