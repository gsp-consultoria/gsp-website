# Prompt 01 — Componentes UI base

Implemente os componentes reutilizáveis da pasta `src/components/ui`.

## Componentes obrigatórios

- `Button.tsx`
- `Card.tsx`
- `Container.tsx`
- `SectionTitle.tsx`

## Requisitos

- Componentes tipados com TypeScript.
- Visual alinhado à paleta da GSP Consultoria.
- Foco visível em elementos interativos.
- Props simples e claras.
- Sem dependência externa de UI.

## Critérios de aceite

- Componentes podem ser usados em qualquer seção.
- Componentes não possuem conteúdo específico de negócio.
- Teste pelo menos um comportamento acessível de botão/link.
