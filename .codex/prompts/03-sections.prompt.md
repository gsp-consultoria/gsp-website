# Prompt 03 — Seções da landing page

Implemente as seções principais em `src/components/sections`.

## Seções obrigatórias

- `Hero.tsx`
- `Services.tsx`
- `Process.tsx`
- `About.tsx`
- `Technologies.tsx`
- `Contact.tsx`

## Copy base

Use um tom profissional, direto e consultivo.

A página deve comunicar:

- Software sob medida.
- Integrações e automações.
- Modernização e evolução tecnológica.
- Diagnóstico antes da construção.
- Entrega incremental e sustentável.

## Critérios de aceite

- Existe apenas um `h1`.
- Cada seção tem `id` para navegação.
- CTAs apontam para contato.
- Seções são responsivas.
- Não há lorem ipsum.
