# Prompt 06 — Testes

Adicione ou melhore testes com Vitest e React Testing Library.

## Prioridades

1. App renderiza seções principais.
2. Header renderiza navegação principal.
3. CTA de contato existe e possui href correto.
4. Componentes UI principais renderizam com nome acessível.

## Restrições

- Não criar snapshots grandes.
- Não testar classe CSS como regra principal.
- Não acoplar teste a detalhes internos.

## Critérios de aceite

- `npm run test` passa.
- Testes são legíveis.
- Testes validam comportamento relevante para usuário.
