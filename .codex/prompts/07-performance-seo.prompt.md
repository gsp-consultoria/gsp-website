# Prompt 07 — Performance e SEO

Revise a aplicação com foco em performance e SEO técnico.

## Verificar

- `index.html` possui `title` e `meta description` adequados.
- Conteúdo principal é claro para mecanismos de busca.
- Não há dependências desnecessárias.
- Imagens futuras devem ser otimizadas.
- Build de produção funciona.

## Melhorias permitidas

- Ajustar metatags.
- Melhorar copy de headings.
- Melhorar semântica HTML.
- Preparar placeholders para Open Graph, sem inventar URL final.

## Critérios de aceite

- `npm run build` passa.
- Metadados refletem a GSP Consultoria.
- Não há dependência nova sem justificativa.
