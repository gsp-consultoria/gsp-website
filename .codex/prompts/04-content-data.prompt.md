# Prompt 04 — Dados e conteúdo estruturado

Organize conteúdo repetível em `src/data`.

## Arquivos

- `src/data/services.ts`
- `src/data/technologies.ts`

## Requisitos

- Tipar os dados com TypeScript.
- Evitar textos genéricos.
- Manter conteúdo alinhado à GSP Consultoria.
- Usar nomes claros e fáceis de editar.

## Critérios de aceite

- Services renderiza dados de `services.ts`.
- Technologies renderiza dados de `technologies.ts`.
- Não há duplicação desnecessária de conteúdo.
