# Prompt 00 — Bootstrap do projeto

Você é um engenheiro front-end sênior especializado em React, TypeScript, Tailwind CSS e Vite.

Crie ou ajuste a fundação da landing page da GSP Consultoria seguindo exatamente as regras abaixo.

## Contexto

A GSP Consultoria desenvolve software sob medida, integrações, automações e consultoria técnica para empresas.

Slogan base:

> Software sob medida para empresas que querem evoluir.

Paleta:

- `#011b7c`
- `#fe701a`
- `#e3f3fe`

## Tarefas

1. Criar projeto React + TypeScript + Vite.
2. Configurar Tailwind CSS.
3. Configurar Vitest + React Testing Library.
4. Criar estrutura de pastas em `src/` conforme `AGENTS.md`.
5. Criar `.env.example`.
6. Criar workflow de CI com lint, test e build.
7. Garantir que `npm run validate` funcione.

## Restrições

- Não adicionar backend.
- Não adicionar CMS.
- Não adicionar bibliotecas de UI.
- Não alterar a paleta definida.

## Critérios de aceite

- `npm run dev` inicia a aplicação.
- `npm run build` gera build de produção.
- `npm run test` executa testes.
- `npm run lint` não retorna erro.
