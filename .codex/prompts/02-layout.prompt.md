# Prompt 02 — Header e Footer

Implemente `Header.tsx` e `Footer.tsx` usando os componentes base.

## Header

Deve conter:

- Marca GSP Consultoria.
- Símbolo visual `>_`.
- Navegação para: Serviços, Processo, Sobre, Tecnologias, Contato.
- CTA para contato.
- Comportamento responsivo.

## Footer

Deve conter:

- Nome da empresa.
- Slogan.
- Links de navegação.
- Estrutura semântica acessível.

## Critérios de aceite

- Navegação usa âncoras estáveis.
- Header é legível em desktop e mobile.
- Footer não duplica informação em excesso.
- Não há links quebrados.
