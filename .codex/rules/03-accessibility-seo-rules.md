# Regras de acessibilidade e SEO

## Acessibilidade

- Deve existir apenas um `h1` na página.
- Cada seção principal deve ter heading descritivo.
- Links e botões devem ter nomes acessíveis claros.
- Foco visível é obrigatório em elementos interativos.
- Não use cor como único meio de transmitir informação.
- Garanta contraste suficiente entre texto e fundo.
- Imagens futuras devem ter `alt` adequado. Imagens decorativas devem usar `alt=""`.

## SEO

- Manter `title` e `meta description` coerentes com software sob medida, consultoria e automação.
- Estrutura de headings deve comunicar a proposta da página.
- Evite conteúdo duplicado ou genérico.
- CTAs devem ser claros e orientados à ação.

## Performance

- Evite dependências pesadas.
- Comprima imagens antes de adicionar em `src/assets`.
- Use SVG para logos e ícones simples quando possível.
- Não introduza animações que prejudiquem Core Web Vitals.
