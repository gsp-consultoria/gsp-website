# Como gerar .rar localmente

Este pacote foi entregue como `.zip` porque o ambiente de geração não possui compactador RAR instalado.

Para gerar `.rar` localmente no Windows:

1. Extraia o `.zip`.
2. Clique com o botão direito na pasta `gsp-consultoria-landing-page`.
3. Selecione **WinRAR > Adicionar para o arquivo...**.
4. Escolha o formato **RAR**.
5. Gere o arquivo.

Nome sugerido:

```txt
gsp-consultoria-landing-page.rar
```
