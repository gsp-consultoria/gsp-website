# Definition of Ready

Uma tarefa está pronta para implementação quando:

- Objetivo está claro.
- Seção ou componente afetado está identificado.
- Copy necessária foi fornecida ou aprovada como rascunho.
- Critérios de aceite estão definidos.
- Impactos em SEO, acessibilidade e responsividade foram considerados.
- Dependências externas foram aprovadas, se houver.
