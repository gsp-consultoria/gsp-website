# Pull Request Checklist

## Descrição

- [ ] Expliquei o que foi alterado.
- [ ] Expliquei por que a alteração foi feita.

## Qualidade

- [ ] Rodei `npm run lint`.
- [ ] Rodei `npm run test`.
- [ ] Rodei `npm run build`.

## UI

- [ ] Validei mobile.
- [ ] Validei desktop.
- [ ] Adicionei prints quando houve mudança visual.

## Acessibilidade e SEO

- [ ] Revisei headings.
- [ ] Revisei foco visível.
- [ ] Revisei links e CTAs.
- [ ] Revisei metadados quando aplicável.
