# Definition of Done

Uma tarefa está concluída quando:

- Código implementado.
- Layout validado em mobile e desktop.
- Acessibilidade básica revisada.
- Testes adicionados ou atualizados quando necessário.
- `npm run validate` executado com sucesso.
- README/specs atualizados quando decisões mudarem.
- Não há placeholders visíveis em produção.
