# Design System Spec

## Direção visual

A identidade deve transmitir:

- Tecnologia.
- Confiança.
- Clareza.
- Modernidade.
- Consultoria premium, mas acessível.

## Paleta

| Token | Hex | Uso |
|---|---|---|
| Brand Blue | `#011b7c` | Fundo institucional, blocos de destaque. |
| Brand Orange | `#fe701a` | CTA, detalhes, indicadores de ação. |
| Brand Sky | `#e3f3fe` | Detalhes claros, textos sobre azul, símbolo `>_`. |

## Componentes base

- Button
- Card
- Container
- SectionTitle

## Regras visuais

- CTAs primários usam laranja.
- Cards devem ter bordas suaves e fundo translúcido ou escuro.
- Seções devem ter espaçamento generoso.
- Evitar excesso de efeitos visuais.
- Preferir layouts claros e com hierarquia forte.

## Tipografia

Usar fonte system UI inicialmente para evitar dependências externas.

Possível evolução futura:

- Inter
- Manrope
- Sora
- Space Grotesk

A escolha final de fonte deve ser documentada em ADR se mudar.
