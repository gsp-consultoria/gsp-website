# Services Spec

## Objetivo

Apresentar claramente os serviços oferecidos.

## Serviços iniciais

- Software sob medida.
- Integrações e automações.
- Modernização de sistemas.
- Consultoria técnica.

## Critérios de aceite

- Serviços vêm de `src/data/services.ts`.
- Cada card tem título e descrição.
- Layout responsivo em grid.
