# About Spec

## Objetivo

Apresentar a GSP Consultoria como uma parceira técnica com visão de negócio.

## Direção de copy

Evitar tom excessivamente pessoal. A seção deve vender a empresa, a capacidade técnica e o método.

## Pontos desejados

- Engenharia de software.
- Arquitetura sustentável.
- Soluções úteis para o negócio.
- Evolução contínua.

## Critérios de aceite

- Texto não parece currículo.
- Reforça confiança.
- Mantém foco em solução para cliente.
