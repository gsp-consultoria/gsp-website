# Contact Spec

## Objetivo

Converter o visitante em conversa comercial.

## Elementos

- Headline orientada à ação.
- Texto curto de convite.
- CTA para WhatsApp.
- CTA para e-mail.

## Critérios de aceite

- WhatsApp usa `VITE_WHATSAPP_NUMBER`.
- E-mail usa `VITE_CONTACT_EMAIL`.
- Mensagem de WhatsApp já vem pré-preenchida.
- CTA é visível no mobile.
