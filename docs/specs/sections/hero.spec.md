# Hero Spec

## Objetivo

Comunicar a proposta de valor em até poucos segundos.

## Mensagem principal

```txt
Software sob medida para empresas que querem evoluir.
```

## Elementos

- Eyebrow de posicionamento.
- `h1` com slogan.
- Texto complementar.
- CTA primário para diagnóstico.
- CTA secundário para serviços.
- Elemento visual tech inspirado em terminal.

## Critérios de aceite

- Existe apenas um `h1`.
- CTA primário abre contato.
- Hero é legível em mobile.
